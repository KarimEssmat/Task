# Mobile Automation QA - Login Functionality

## Project Description
This project automates the login functionality of the My Demo App for both Android and iOS platforms. The script verifies successful login and various failure scenarios using Appium and WebDriverIO with Cucumber BDD.

## Prerequisites
1. Node.js (>=14.x)
2. Appium installed globally (`npm install -g appium`)
3. WebDriverIO CLI installed (`npm install -g @wdio/cli`)
4. Android/iOS Emulator or real device configured
5. Java Development Kit (JDK) installed
6. Dependencies: Install them with `npm install`
